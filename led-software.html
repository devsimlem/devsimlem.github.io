QA LED Tester

Contracted Software

The client for this project is a large aerospace company that delivers solutions for major airlines. This client wanted software that could utilize user-entered configuration files to determine whether LED test results met the standards they determined. I communicated daily with the client and worked effectively to deliver the project within a strict time frame. I was the sole contributor for this program, which is expected to save the company ~$20,000 annually.

This project allowed me to develop time management skills on a larger scale. I broke the project down into smaller pieces and had deadlines set for each milestone within the project to ensure the project was delivered on time. Additionally, I was able to improve my software development skills in Java, which I previously had only used in school. Lastly, since this was my first solo contracted project, I learned how to effectively communicate with clients and understand their desires before starting to write any code.
