Rob's Trek

A Retro-Inspired Platformer

Play the game HERE!

Rob's Trek is a pixel adventure game where you, Rob, must find and eliminate your target, Mr. Pickles. Run, shoot and jump your way through this planet, but make sure to be ammo-efficient! The idea behind this game was to incorporate the concept of an action game with that of one where you have to manage your ammunition throughout the level, giving the game more strategy. There are various paths to take and I found that this game provides replayability and a fun game to challenge your friends in time trials or other sorts. This project was developed for a class using construct3, a web-based software for creating games.

Skills developed: Construct3, project management, pixel art, game balancing and game design, level design, sound and music incorporation.
